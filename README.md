## Run Locally

Clone the project

```bash
  git clone https://github.com/fredysiswanto/foodtrip-api
```

Go to the project directory

```bash
  cd foodtrip-api
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
